# To-Do List App 

Simple task manager built with **React**, **Vite** and **Tailwind CSS**.

This project was created for learning purposes, focusing on React fundamentals, controlled inputs, state management, and Git workflow using feature branches.

---

## Features

- Add new tasks

- Edit existing tasks

- Delete tasks

- Mark tasks as completed

- Filter tasks:
    - All
    - Pending
    - Completed

- Responsive design (mobile & desktop)

---

## Tech Stack

- **React**
- **Vite**
- **Tailwind CSS**
- **JavaScript (ES6+)**

---

## Installation

Clone the repository and install dependencies:

```bash
npm install
npm run dev
```
